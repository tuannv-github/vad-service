Parameters

Active values saved to data/vad_settings.json. Reset / Reset defaults restore from config/vad_defaults.json.

Silero VAD

  • threshold

    Speech probability threshold (0–1). Silero scores each 512-sample window; windows at or above this value are treated as speech. Higher values reduce false positives but may miss quiet speech. Silero library default: 0.5; this service default: 0.9.

  • min_speech_ms

    Minimum speech segment length in milliseconds. Shorter detections are discarded as noise or clicks. Silero parameter: min_speech_duration_ms. Default: 500.

  • min_silence_ms

    Minimum trailing silence in milliseconds before voice_activity_end (utterance complete). Silero parameter: min_silence_duration_ms. Default: 1000.

  • speech_pad_ms

    Padding in milliseconds added around each detected speech segment so word onsets and endings are not clipped. Silero parameter: speech_pad_ms. Default: 100.

  • neg_threshold

    Exit threshold for ending speech. When probability drops below this value, Silero looks for silence to close the segment. Leave empty to use Silero’s default: threshold − 0.15 (minimum 0.01).

  • max_speech_duration_s

    Maximum continuous speech duration in seconds before Silero forces a segment split (useful for very long monologues). Leave empty for no limit. Silero parameter: max_speech_duration_s.
